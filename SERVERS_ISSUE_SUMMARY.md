# Проблема с серверами

## Текущий статус

### Backend
- ✅ **Слушает** на порту 8000 (процесс 12380)
- ❌ **Возвращает 502** при запросах
- ⚠️ **Проблема**: Сервер запущен, но не обрабатывает запросы правильно

### Frontend
- ❌ **Не слушает** на порту 5173
- ✅ **Node процесс** запущен (12772)
- ⚠️ **Проблема**: Vite не запустился или запустился на другом порту

## Решение

### 1. Перезапустить Backend
```bash
# Остановить
taskkill /F /PID 12380

# Запустить заново
cd C:\Users\User\Desktop\rag2\backend
python run.py
```

### 2. Запустить Frontend вручную
```bash
cd C:\Users\User\Desktop\rag2\frontend
npm run dev
```

Смотрите вывод в консоли - там будет указан порт и возможные ошибки.

### 3. Проверить ошибки

**Backend:**
- Проверьте логи: `backend/logs/backend.log`
- Ищите ошибки импорта или конфигурации

**Frontend:**
- Проверьте консоль терминала
- Убедитесь, что `node_modules` установлены: `npm install`

## Альтернатива
