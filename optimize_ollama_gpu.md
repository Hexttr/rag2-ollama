# Оптимизация Ollama для RTX 5060 8GB

## Настройки для GPU

### 1. Проверка использования GPU

Ollama автоматически использует GPU, если доступен CUDA. Проверьте:
```bash
ollama ps
```

### 2. Рекомендуемая модель

Для RTX 5060 8GB рекомендуется:
- **phi3:3.8b** - быстрая и эффективная модель (~2GB)
- **llama3.1:8b** - более качественная, но медленнее (~4.5GB)

### 3. Настройки приложения

- ✅ Модель изменена на `phi3:3.8b` для лучшей производительности
- ✅ Уменьшены параметры PageIndex для более быстрой обработки
